# :lucide-hammer: Sideloading Tools

## PC

- [**⭐ Rookie (Official)**](https://github.com/nerdunit/androidsideloader/) 
    
    The original sideloading tool. Requires a server. See [the guide](https://github.com/CrazyScamp/Rookie-3.1-Setup/) for setup with the new JSON.
    
    
- [**⭐ ApprenticeVrSrc**](https://github.com/KaladinDMP/apprenticeVrSrc)
    
    Improved fork of ApprenticeVR. Also requires a server. Follow [the guide](https://github.com/KaladinDMP/apprenticeVrSrc#option-a-in-app-recommended) to add the latest JSON.

- [**VRSideForge**](https://github.com/yGuilhermy/VRSideForge)
   
    Indexer for RuTracker. Downloads games via torrents.

- [**QLoader**](https://github.com/skrimix/QLoaderFiles/)
   
    Alternative sideloader. Requires manual server setup.

- [**LFI Sideloader**](https://github.com/LordSpectre/LFI-Sideload-Tool)
   
    Alternative sideloader. Lightweight APK and OBB sideloader.

- [**Quest File Transfer (QFT)**](https://github.com/extremelystiff/Oculus-Quest-File-Transfer-Tool/)

    Alternative sideloader for manual file transfer using a dual-pane file manager.

- [**QuestVault**](https://github.com/ezi0n/QuestVault)
    
    QuestVault is a multi-platform AIO desktop Quest content manager for local library indexing, headset inventory, save backup workflows, ADB operations, and vrSrc-plugin remote catalog downloads.

### Web

!!! warning "These require a browser that supports WebUSB."
    Currently this is only available in Chromium-based browsers (Chrome, Edge, Opera, etc).

    Safari and Firefox are not supported.

- [**Quest Webloader**](https://webloader.app/)
   
    Browser-based sideloading tool for Quest.

- [**QuestLoad**](https://frizzlem.github.io/QuestLoad/)

    Simple browser-based sideloading utility.

- [**Web ADB Menu**](http://twhlynch.me/Web-ADB-Menu/)

    A web-based ADB tool for Meta Quest. Does much more than sideloading, including access to various Quest performance settings.

### Other

- [**APC (Automagic Package Changer)**](https://github.com/KaladinDMP/AutomagicPackageChanger)

    A desktop utility that renames Quest APK package names for easier app management and conflict avoidance. It handles the package-changing process automatically with a simple drag-and-drop workflow.
