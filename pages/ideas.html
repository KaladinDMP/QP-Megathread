Idea for Working Compatibility List :
I’ll be testing quite a lot of Quest APKs on firmware 201 over the coming months. Optimistically - 2,680 to be precise (O.o)

I’ve started creating notes.txt files for each game, indicating the test status.

Some crash after compiling shaders, others can’t install due to a lower target SDK version, while some work straight out of the box (even ones that are labeled v74).

It might be a good idea to create a space here where I can post these notes for each game, including the firmware version and the full game release name. This way, people will have a reference for whether an older v74 game works on newer firmware, and so on.

If other community members join in, we could eventually build something similar to the compatibility lists seen with emulators, largely community-driven and continuously improved over time.
