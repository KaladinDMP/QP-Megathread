Community-made standalone tools for Quest users, including PC sideloaders, browser-based loaders, and other useful utilities.

💻 PC Sideloaders

These Desktop applications sideload Quest content to your headset. They do not include a built-in server — you must configure one separately. For sideloader with servers, see Quest Standalone Sources.

  • Rookie (Official) Windows Only: The original sideloading tool. Requires a server. See the public JSON page for setup. Download
  • LFI Sideloader: Multi-OS: Lightweight APK/OBB sideloader built in Python. Should work anywhere Python runs. GitHub
  • Quest File Transfer (QFT): Multi-OS: Dual-pane file manager for manual transfer. Python-based. GitHub

Note: The above sideloaders require a server configuration for the included gamelist function. The easiest method is via the public JSON. For a full walkthrough, see Rookie 3.2 Setup Guide or refer to the specific sideloader's setup guide.

🌐 WebLoaders

These browser-based tools allow direct sideloading or Quest management through WebUSB. Requires a supported browser such as Chrome, Edge, Opera, or Brave.

  • Quest Webloader: Browser-based sideloading tool for Quest. Website
  • QuestLoad: Simple browser-based sideloading utility. Website
  • Web ADB Menu: A web-based ADB tool for Meta Quest. Does much more than sideloading, including access to various Quest performance settings. GitHub

Note: These will not work in Firefox or Safari because those browsers do not support WebUSB.

🎮 Native Quest Tools

Native Quest applications and utilities that run directly on the headset without requiring a PC companion app.

  • Evolve Launcher Quest Native: Native Quest launcher replacement focused on customization, organization, and improved library management directly inside the headset. GitHub
  • Lightning Launcher Quest Native: Lightweight custom launcher for Quest with improved app organization, shortcuts, categories, and cleaner navigation compared to the stock Meta launcher. GitHub
  • Quest Games Optimizer (QGO) Quest Native: Performance and visual tuning utility for Quest games and applications. Allows changing resolution, refresh rate, GPU/CPU profiles, and more. Website
  • CX File Explorer Quest Native: Android-based file manager commonly used on Quest for browsing local storage, managing APKs, and handling headset files directly on-device. This also allows FTP login (and other network login methods), convenient for sites like 3DFlickFix.net and other FTP client-based services. Website

Note: Most native Quest tools may require Developer Mode enabled for installation or advanced functionality.

🧩 General Standalone Tools

Other standalone utilities that do not fit directly into the sideloader categories but that require a PC.

  • Quest Game Patcher Windows Only: A desktop application to patch standalone Quest games to bypasses firmware version checks, allowing games to run without forced headset updates. Original GitHub release was taken down, but an info thread on Reddit exists and an alternative download mirror are now available, with users encouraged to report results on the the reddit thread.
